# Запуск клиентской части

1. перейти в папку клиента - 'cd client'
2. инициализировать все npm пакеты - 'npm i'
3. запустить клиент - 'npm start'

# Для получения данных с сервера

1. Запускаем в отдельном терминале серверную часть

- Перейти в папку сервера часть - 'cd server'
- Запустить сервер - 'npm run dev'

2. В отдельном терминале запустить клинент

- перейти в папку клиента - 'cd client'
- запустить клиент - 'npm start'

###

Клиент работает на [http://localhost:3000]
